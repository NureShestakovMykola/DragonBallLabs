# DragonBallLabs

2D-проєкт на Unity з хвилями ворогів і босами, системою здоров’я, візуальними ефектами, звуками та керуванням аудіо.

---

## Коротко про гру
Гравець керує персонажем у 2D-просторі, стріляє по ворогах, виживає хвилі та періодично зустрічає босів зі збільшеним запасом HP. Є відображення HP, нарахування очок, система Game Over, стартовий екран, звукові ефекти та фонова музика.

---

## Реалізовано
- Рух гравця з інерцією, орієнтація за курсором
- Стрільба, параметри: швидкість, шкода, lifetime
- Вороги: переслідування гравц
- Хвилі ворогів із наростанням складності
- Боси кожні 3 хвилі
- Ефекти смерті через Particle System
- Ауді
- Кнопка вмикання/вимикання звуку

---

## Управління
- Рух: **W/A/S/D**
- Постріл: **Mouse0** / **Space**
- Перезапуск після поразки: **R**
- Звук: кнопка **Sound ON/OFF** на екрані

---

## Запуск збірки (Build)
1. Відкрити розділ **Releases** у репозиторії.
2. Завантажити архів **Windows Build (.zip)**.
3. Розпакувати архів у будь-яку папку.
4. Запустити файл `*.exe`.

> Якщо збірка не стартує: перевірити, що поруч із `.exe` лежить папка `*_Data` та всі `.dll`, які були в архіві.

---

## Запуск проєкту в Unity
1. Unity версія: **<Unity 6.3 LTS>**
2. Відкрити папку проєкту через Unity Hub → **Open**.
3. Відкрити сцену: `Assets/Scenes/SampleScene.unity`
4. Натиснути **Play**.

---

## Структура проєкту 
- `Assets/`
  - `Scripts/` — основні скрипти керування грою  
  - `Audio/` — SFX та музика  
  - `FX/` — префаби ефектів (смерть ворога/боса тощо)  
  - `Scenes/` — сцени
- `Builds/` — локальна збірка (не обов’язково зберігати в git; краще як Release)

---

## Ключові скрипти
- `PlayerScript.cs` — рух, стрільба, виклик звуку пострілу.
- `PlayerHealth.cs` — HP гравця, отримання шкоди, синхронізація UI, GameOver.
- `EnemyHealth.cs` — HP ворога/боса, смерть, FX смерті, нарахування очок.
- `EnemyRandomMove.cs` — поведінка ворогів (переслідування/випадковість/відштовхування/межі).
- `EnemySpawner.cs` — хвилі, спавн ворогів і босів, баланс.
- `BossHPBar.cs` — HP-bar над босом.
- `AudioManager.cs` — музика, SFX, гучність, mute.
- `SoundToggleButton.cs` — кнопка ON/OFF звуку.
- `GameManager.cs` — score, wave, game over, рестарт.
